LazyFunction

发表于 2021-06-03 更新于 2021-06-22 分类于 javascript ， function

普通函数

1. timeStamp 污染全局
2. 执行第一次之后，timeStamp 已经有值，但后续每次执行都会进入 if 语句

惰性函数

1.执行一遍外层函数，拿到需要的值后，在函数内部重写自身，返回需要的值
2.后续调用 不需要每次都执行原函数 重读的代码段

实际应用

普通函数添加事件

每次执行都要对浏览器进行判断。 可以在第一次判断后就保留判断结果，再次执行时就不需要判断了。

惰性函数添加事件

第一次判断之后，在函数体内部重写自身，后续执行的是重写后的函数，不需要再进行无用的判断

debounce & throttle

发表于 2021-06-02 更新于 2021-06-03 分类于 javascript ， function

函数防抖

n秒内只要触发事件，就重新计时，
一直触发事件处理函数的程序永远不能被执行

函数节流

事件被触发，n秒之内只执行一次事件处理函数
